# NEON tips

## 本番環境の DB を簡易バックアップする

> ローカルで手動で行う方法

### 1️⃣ ターミナルでコマンドを叩く

👉 *ローカルの `backup.sql` ファイルにバックアップ*

```bash
pg_dump "DATABASE_URL" > backup.sql
```

  ⬇️

実行後、`backup.sql` が作成されるので `.gitignore` に追加

```ruby
# .gitignore
backup.sql
```

---

### 2️⃣ コマンドをカスタマイズする

#### バックアップファイルに日付を添える

```bash
pg_dump "DATABASE_URL" > backup_$(date +%F).sql
```

```ruby
# .gitignore
backup_*.sql
```

#### コマンドをエイリアス化する

```bash
alias backupdb='pg_dump "DATABASE_URL" > backup_$(date +%F).sql'
```

👉 *次から `backupdb` でバックアップが実行される`

---

### 3️⃣ 古いバックアップを消す

```bash
find . -name "backup_*.sql" -mtime +7 -delete
```

👉 *日付つきファイルで管理する場合は、一定期間ごとに不要ファイルを整理しておきたい（上記は「7日より古いファイルを削除」する例）*

---

### 4️⃣ バックアップから復元する

```bash
# 2026-03-26 のデータを復元する場合
psql "DATABASE_URL" < backup_2026-03-26.sql
```

⚠️ バックアップからの復元は既存データを上書きするので要注意

対策：

- 別 DB に復元して確認 *（NEON に新しいブランチをつくって確認する）*
- ローカル DB に復元して確認 *（ローカルに PostgreSQL が必須❗️）*
