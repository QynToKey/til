# JavaScript tips

## スライダーの操作

HTML の例:

```html
<label>
  <span>文字サイズ (<span id="font-size-label">16</span>)</span>
  <input type="range" min="14" max="24" id="font-size" value="16">
</label>
```

⬇️

```js
const slider = document.getElementById('font-size');

slider.addEventListener('input', () => {
  const fontS = slider.value;
});
```

- `type="range"` の値(**フォーム要素の現在値**)は `.value` で取得する
- `type="range"` のイベント操作(**リアルタイム反映したい場合**)は `input` イベントを使う

👉 *`text` / `textarea` / `number` も同様*

---

## クラス名に `.` をつける？ つけない？

- セレクタを書く時は `.` が必要

```js
document.querySelector('.current');
```

👉 *セレクタは「CSS の文法」に則る*

- `classList` を操作する時は `.` 不要

```js
element.classList.remove('current');
```

👉 *`classList` には「クラス名そのもの」を書く*
