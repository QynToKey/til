# [卒制](https://github.com/QynToKey/HowLongWillItLast) (MVP 後)： ログイン機能のリファクタリング

## 0️⃣ ユーザーさんからのフィードバック

```text
アプリの使い心地ですが、毎回ログインするのが面倒で‥
あアレ、パスワードだけになるとものすごく楽になるかと‥
```

  ➡️ Sorcery の `remember_me` 機能を使用して対応する

---

## 1️⃣ Sorcery `remember_me` を有効化

### `config/initializers/sorcery.rb`

```ruby
# Rails.application.config.sorcery.submodules = [:user]
```

  ⬇️ コメントアウトを解除して `remember_me` を設定

```ruby
Rails.application.config.sorcery.submodules = [:remember_me]
```

---

## 2️⃣ マイグレーションを生成

```bash
$ docker compose exec web rails g migration AddRememberMeTokenToUsers \
>   remember_me_token:string \
>   remember_me_token_expires_at:datetime
      invoke  active_record
      create    db/migrate/20260329024027_add_remember_me_token_to_users.rb
```

  ⬇️

```ruby

```
