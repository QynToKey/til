## ビューの実装

### 1️⃣ 一覧ページ

```bash
touch app/views/tags/index.html.erb
```

```ruby
<h1>タグ一覧</h1>

<%= link_to 'タグを追加する', new_tag_path %>

<table>
  <thead>
    <tr>
      <th>タグ名</th>
    </tr>
  </thead>
  <tbody>
    <% @tags.each do |tag| %>
      <tr>
        <td><%= tag.name %></td>
        <td>
          <%= link_to '編集', edit_tag_path(tag) %>
          <%= link_to '削除', tag_path(tag), data: { turbo_method: :delete, turbo_confirm: '本当に削除しますか？' } %>
        </td>
      </tr>
    <% end %>
  </tbody>
</table>
```

---

### 2️⃣ 作成ページ

```bash
touch app/views/tags/new.html.erb
```

```ruby
<h1>タグの作成</h1>

<% if @tag.errors.any? %>
  <div style="color: red;">
    <h2><%= @tag.errors.count %>件のエラーがあります</h2>
    <ul>
      <% @tag.errors.full_messages.each do |message| %>
        <li><%= message %></li>
      <% end %>
    </ul>
  </div>
<% end %>

<div>
  <%= form_with model: @tag, do |f| %>
    <div>
      <%= f.label :name, "タグ名" %><br>
      <%= f.text_field :name %>
    </div>

    <div>
      <%= f.submit "作成する" %>
      <%= link_to "戻る", tags_path %>
    </div>
  <% end %>
</div>
```

### 4️⃣ 共通部分をパーシャル化

```bash
touch app/views/tags/_form.html.erb
```

```ruby
<div>
  <%= form_with model: @tag do |f| %>
    <div>
      <%= f.label :name, "タグ名" %><br>
      <%= f.text_field :name %>
    </div>

    <div>
      <%# f.submit のラベルはi18n対応後に削除予定 %>
      <%= f.submit tag.new_record? ? "記録する" : "更新する" %>
      <%= link_to "戻る", return_path %>
    </div>
  <% end %>
</div>
```

⬇️ `new` と `edit` の当該箇所を以下と差し替え

```ruby
<%= render 'form', tag: @tag, return_path: tags_path %>
```
