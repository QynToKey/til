## 0️⃣ 現状と実装方針

### 現状

```ruby
# app/controllers/users_controller.rb
  def create
    @user = User.new(user_params) # ⬅️ user_params は email, password, password_confirmation のみ
    ・・・
  end
```

👉 *`role` が `params` に含まれないため、DB のデフォルト値 0（= member）で作成される。つまり 現状は誰でも `/register` から一般ユーザー（member）として登録できる 状態*

### 実装方針

- admin ロール付きアカウントを、 `/register` からではなく、使い捨てトークン付き URL `/admin/register?token=xxx` 経由でのみ作成可能にする。
- トークンは `seed` で生成し、使用後に無効化する。
- 現状の `/register` は member アカウント専用のまま残す。

## 2️⃣ `AdminRegistrationToken` モデルを作成

```bash
touch app/models/admin_registration_token.rb
```

```ruby
# app/models/admin_registration_token.rb
class AdminRegistrationToken < ApplicationRecord
  # 管理者登録用のトークンモデル
  before_create :generate_token

  # トークンがまだ使用されていないかを確認するメソッド
  def available?
    used_at.nil?
  end

  # トークンを使用済みにするメソッド
  def use!
    update!(used_at: Time.current)
  end

  private

  # トークンを生成するメソッド
  def generate_token
    self.token = SecureRandom.urlsafe_base64(32)
  end
end
```

## 3️⃣ `Registrations` コントローラーを作成

```bash
mkdir app/controllers/admin/
touch app/controllers/admin/registrations_controller.rb
```

```ruby
# app/controllers/admin/registrations_controller.rb
class Admin::RegistrationsController < ApplicationController
  # 管理者登録はトークンを使用して行うため、ログインを要求しない
  skip_before_action :require_login

  def new
    @token_record = find_available_token
    unless @token_record
      redirect_to new_session_path, alert: "無効または使用済みのトークンです"
      return
    end
    @user = User.new
  end

  def create
    @token_record = find_available_token
    unless @token_record
      redirect_to new_session_path, alert: "無効または使用済みのトークンです"
      return
    end

    @user = User.new(user_params.merge(role: :admin))
    if @user.save
      @token_record.use!
      auto_login(@user)
      redirect_to root_path, notice: "管理者アカウントを作成しました"
    else
      render :new, status: :unprocessable_entity
    end
  end

  private

  # トークンが有効で使用されていないかを確認し、利用可能なトークンを返すメソッド
  def find_available_token
    record = AdminRegistrationToken.find_by(token: params[:token])
    # もしトークンが存在し、かつ利用可能であればそのレコードを返し、そうでなければnilを返す
    record&.available? ? record : nil
  end

  # Strong Parametersを使用して、ユーザーの入力を安全に処理するためのメソッド
  def user_params
    params.require(:user).permit(:email, :password, :password_confirmation)
  end
end
```
