# [co-READER](https://github.com/QynToKey/co_reader)（day: 14_3）： テキスト UI

## 1️⃣ ルーティング

`texts` を `reading_sessions` の下にネストする

```ruby
# config/routes.rb
  resources :reading_sessions, only: %i[ index show ] do
    get :join, on: :collection
    resources :texts, only: %i[ show ]
```

👉 *URL は `/reading_sessions/:reading_session_id/texts/:id` になる。

---

## 2️⃣ `Texts` コントローラーに `show` アクションを追加

```ruby
class TextsController < ApplicationController
  # ログインユーザーにテキストを表示するためのコントローラー
  before_action :require_login
  before_action :set_text
  before_action :set_reading_session
  before_action :require_membership


  def show
    session_text = @reading_session.tests.order(:id) # #95 で position カラムを追加したら order(:position) に変更
    current_index = session_text.index(@text)
    @prev_text = session_text[current_index - 1] if current_index > 0
    @next_text = session_text[current_index + 1]
  end

  private

  # テキストをセットするためのコールバック
  def set_text
    @text = Text.find(params[:id])
  end

  # リーディングセッションをセットするためのコールバック
  def set_reading_session
    @reading_session = ReadingSession.find(params[:reading_session_id])
  end

  # セッションへの参加を確認するためのコールバック
  def require_membership
    unless @reading_session.members.include?(current_user)
      redirect_to reading_sessions_path, alert: "参加していないセッションです"
    end
  end
end
```
