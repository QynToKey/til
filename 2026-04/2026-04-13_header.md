# [co-READER](https://github.com/QynToKey/co_reader)（day: 7_2）： ヘッダー

## 概要

- タイトルを co-READER に修正
- ヘッダーを導入

### 実装内容

- `application-name` を co-READER に変更
- `app/views/shared/_header.html.erb` を作成
- Bootstrap のクラスを flash メッセージのタイプに対応
- `sessions/new.html.erb` のアラート表示の重複を削除してヘッダー表示に統一

---

### Rails の flash タイプと Bootstrap のクラスの対応

| flash type | Bootstrap class |
| --- | --- |
| `notice` | `alert-success` |
| `alert` | `alert-danger` |
| `info` | `alert-info` |
| `warning` | `alert-warning` |
