# [HLWIL](https://github.com/QynToKey/HowLongWillItLast) : ヒートマップ実装

## 0️⃣ 実装方針

- `UsersController#show` にヒートマップ用の集計クエリを追加
- タグ集計（`@tag_summaries_by_theme`）は既存のものをそのまま流用
- Bootstrap の `collapse` コンポーネントでアコーディオンを実現 → 実装後に削除・常時表示に変更
- CSS Grid の `grid-auto-flow: column` でカラムが「週」になる GitHub 風レイアウト

---

## 1️⃣ `UsersController#show` にヒートマップ用のロジックを追加

```ruby
# app/controllers/users_controller.rb
  def show
    @learning_themes = current_user.learning_themes

    # 各 learning_theme ごとにタグの学習時間を計算してまとめる
    @tag_summaries_by_theme = @learning_themes.each_with_object({}) do |theme, hash|
      hash[theme.id] = theme.tags.map do |tag|
        {
          name: tag.name,
          hours: (current_user.total_learning_minutes_by_tag(tag) / 60.0).round(1)
        }
      end
    end

+   # ヒートマップ用：テーマごとに過去1年の日別学習時間（分）を集計する
+   @heatmap_data_by_theme = @learning_themes.each_with_object({}) do |theme, hash|
+     hash[theme.id] = theme.learning_records
+       .where(study_date: 1.year.ago.to_date..)
+       .group(:study_date)
+       .sum(:duration_minutes)
+   end
  end
```

## 3️⃣ 「ヒートマップ」ビューを作成

```bash
touch app/views/users/_heatmap.html.erb
```

```erb
<% start_date = 1.year.ago.to_date %>
<% end_date = Date.today %>
<% grid_start = start_date - start_date.wday %>

<%# 週ごとの配列を構築（月ラベル用） %>
<% weeks = [] %>
<% w = grid_start %>
<% while w <= end_date %>
  <% weeks << w %>
  <% w += 7 %>
<% end %>

<div class="heatmap-wrapper" data-controller="heatmap">
  <%# 月ラベル %>
  <div class="heatmap-months">
    <% weeks.each do |week_start| %>
      <% week_end = [week_start + 6, end_date].min %>
      <% month_start = (week_start..week_end).find { |d| d.day == 1 } %>
      <div class="heatmap-month-label"><%= month_start&.strftime('%-m月') %></div>
    <% end %>
  </div>

  <%# ヒートマップグリッド %>
  <div class="heatmap-grid">
    <% current_date = grid_start %>
    <% while current_date <= end_date %>
      <% minutes = heatmap_data[current_date] || 0 %>
      <% level = case minutes
                 when 0       then 0
                 when 1..30   then 1
                 when 31..90  then 2
                 when 91..180 then 3
                 else              4
                 end %>
      <div class="heat-cell heat-<%= level %>"
           title="<%= current_date.strftime('%Y/%m/%d') %> : <%= (minutes / 60.0).round(1) %>h"></div>
      <% current_date += 1 %>
    <% end %>
  </div>
</div>
<p class="text-end text-muted small mt-1">過去1年間で <%= heatmap_data.count %> 日学習しました</p>
```

📝 `grid_start = start_date - start_date.wday` ：
「その週の日曜日」に遡り、グリッドを列（週）単位で揃える。

📝 時間（分）からレベルを決める閾値：

```erb
<% level = case minutes
           when 0       then 0   # 記録なし
           when 1..30   then 1   # 〜30分
           when 31..90  then 2   # 〜1.5時間
           when 91..180 then 3   # 〜3時間
           else              4   # 3時間超
           end %>
```

## 4️⃣ CSS にヒートマップ描画の設定を追加

```css
.heatmap-wrapper {
  overflow-x: auto;
}

.heatmap-months {
  display: flex;
  margin-bottom: 4px;
  width: max-content;
}

.heatmap-month-label {
  flex-shrink: 0;
  width: 15px;
  font-size: 0.7rem;
  color: #6e7681;
  white-space: nowrap;
  overflow: visible;
}

/* グリッドの構造 */
.heatmap-grid {
  display: grid;
  grid-template-rows: repeat(7, 12px); /* 7行（日〜土）固定 */
  grid-auto-flow: column;              /* セルを列（週）方向に積む */
  gap: 3px;                            /* セル間の隙間 */
  overflow-x: auto;  /* 冗長・width: max-content により実質無効 */
  width: max-content;
}

/* セルの形 */
.heat-cell {
  width: 12px;
  height: 12px;
  border-radius: 2px; /* 角丸 */
}

/* 学習時間ごとの色 */
.heat-0 { background-color: #ebedf0; } /* 0分：グレー */
.heat-1 { background-color: #9be9a8; } /* 1〜30分：薄緑 */
.heat-2 { background-color: #40c463; } /* 31〜90分：緑 */
.heat-3 { background-color: #30a14e; } /* 91〜180分：濃緑 */
.heat-4 { background-color: #216e39; } /* 181分〜：最濃緑 */
```

## 5️⃣ `heatmap_controller.js` を作成

接続時に右端（最新日）へ自動スクロールさせる。

```bash
touch app/javascript/controllers/heatmap_controller.js
```

```js
import { Controller } from "@hotwired/stimulus"

export default class extends Controller {
  connect() {
    requestAnimationFrame(() => requestAnimationFrame(() => {
      this.element.scrollLeft = this.element.scrollWidth
    }))
  }
}
```

👉 *`this.element` は `heatmap-wrapper` を指す。*
👉 *ダブル `requestAnimationFrame` で2フレーム待つことでブラウザのレイアウト計算完了後にスクロール*
