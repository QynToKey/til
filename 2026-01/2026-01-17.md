- 「三択クイズ」アプリの完成

  - 問題文を配列に格納する
    ```
    const quizzez = [
      ['1の正解は？', '選択肢 A', '選択肢 B', '選択肢 C', 0],
      ['2の正解は？', '選択肢 A', '選択肢 B', '選択肢 C', 1],
      ['3の正解は？', '選択肢 A', '選択肢 B', '選択肢 C', 2],
    ];
    ```

  - 出題処理を`forEach()`でまとめる
    ```
    quizzez.forEach((quiz) => {
      render(quiz);
    });
    ```
      └ `render関数`は別途用意する: `createElement()`と`appendChild()`を活用
