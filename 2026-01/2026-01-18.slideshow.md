## JavaScriptで「[スライドショー](https://github.com/QynToKey/MySlideshow)」を試作する

### `CSS` tips
- `vertical-align: bottom;`: インラインボックス内で、画像の底を`img`要素の行ボックスの底に合わせて配置する(または、`baseline`からの距離を指定できる)

### `JavaScript`の実装について
- `forEach`は引数の順番で意味が決まっている。

  - 基本形
    ```
    配列.forEach((要素, インデックス, 配列自身) => {
      // 処理
    });
    ```

  - 引数の意味
    | 順番   | 引数      | 意味           |
    | ---- | ------- | ------------ |
    | 第1引数 | element | 現在の要素        |
    | 第2引数 | index   | 現在のインデックス    |
    | 第3引数 | array   | 元の配列（ほぼ使わない） |

  - 要素`element`だけ使うとき（多くの場合これ）
    ```
    thumbnails.forEach(thumbnail => {
      thumbnail.classList.remove('active');
    });
    ```

  - インデックス`index`だけ使うとき（あまりない）
    ```
    thumbnails.forEach((_, index) => {
      console.log(index);
    });
    ```
      └ 使わない引数は慣習的に`_`で表す

  - 配列`array`は`forEach`を呼び出した配列そのもの（単独で使うことはほぼ無い）
    ```
    # 例
      items.forEach((item, index, array) => {
        if (array.length === 1) {
          console.log('1件しかない');
        }
      });
    ```
