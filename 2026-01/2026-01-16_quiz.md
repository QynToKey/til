- JavaScriptで「三択クイズ」アプリを試作

  - `html`要素を動的に生成する。
    ```
    # 親要素を取得
    const main = document.querySelector('main');
    # 追加する要素を生成
    const section = document.createElement('section');
    # 子要素として追加
    main.appendChild(section);
    ```

  - 要素は下位のものから積み上げる。
    ```
    section.appendChild(h2); # 生成した子要素をdocument外に追加
    main.appendChild(section); # documentツリーに接続
    ```
      - 親要素を先に`documentツリー`に接続してしまうと、その後に`DOM`変更が生じてしまい、若干だがブラウザに負荷がかかる。
      - 子要素を先に生成してから`documentツリー`に接続する順番が推奨される。

  - `classList.add()`の実行結果について誤解していたこと
    ```
    li.classList.add('correct');
    ↓
    <li class="correct">

    ⭕️ li.correct # 複合セレクタ: li自身にcorrectクラスが付与される
    ❌ li .correct # 子孫セレクタ: liの子孫にcorrectクラスが付与されるわけではない
    ```
