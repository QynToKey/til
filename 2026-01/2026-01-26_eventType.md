# イベントタイプ一覧

## 基本構文
  ```
  addEventListener(type, listener)
  ```

  - `addEventListener`: イベントリスナーを登録するメソッド
  - `type`: どのイベントを監視するかを指定する文字列
  - listner `() => {}`: イベント発生時に実行されるイベントハンドラ（イベントリスナー関数）


### マウス関連イベント
  | イベントタイプ      | 意味                  |
  | ------------ | ------------------- |
  | `click`      | クリックされた             |
  | `dblclick`   | ダブルクリックされた          |
  | `mousedown`  | マウスボタンを押した          |
  | `mouseup`    | マウスボタンを離した          |
  | `mousemove`  | マウスを動かした            |
  | `mouseenter` | 要素にマウスが入った（バブリングなし） |
  | `mouseleave` | 要素からマウスが出た（バブリングなし） |
  | `mouseover`  | 要素にマウスが入った          |
  | `mouseout`   | 要素からマウスが出た          |

### キーボード関連イベント
  | イベントタイプ    | 意味       |
  | ---------- | -------- |
  | `keydown`  | キーを押した瞬間 |
  | `keyup`    | キーを離した瞬間 |
  | `keypress` | ※非推奨（古い） |

### フォーム・入力関連イベント
  | イベントタイプ  | 意味           |
  | -------- | ------------ |
  | `change` | 値が変更され、確定した  |
  | `input`  | 値が変更されるたびに発火 |
  | `submit` | フォームが送信された   |
  | `focus`  | フォーカスされた     |
  | `blur`   | フォーカスが外れた    |

### タッチイベント（スマホ向け）
  | イベントタイプ      | 意味     |
  | ------------ | ------ |
  | `touchstart` | 画面に触れた |
  | `touchmove`  | 指を動かした |
  | `touchend`   | 指を離した  |

### ウィンドウ、ドキュメント関連（ページ全体の状況変化)
  | イベントタイプ            | 意味            |
  | ------------------ | ------------- |
  | `load`             | ページ・画像の読み込み完了 |
  | `DOMContentLoaded` | HTMLの解析完了     |
  | `resize`           | ウィンドウサイズ変更    |
  | `scroll`           | スクロールされた      |
