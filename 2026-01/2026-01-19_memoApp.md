## JavaScriptで「[メモ帳](https://github.com/QynToKey/MyMemoApp)」を試作する

### CSS tips

- `justify-content`: フレックスボックスの主軸方向(← →)の揃え
- `align-items`: フレックスボックスの交差軸方向(↑ ↓)の揃え
- `margin-left: auto;`: `margin`における`auto`はその値が最大になるので、結果的に「右寄せ」で表示される

### `JavaScript`の実装について

- `localStorage`: ブラウザに用意されている機能で、`JavaScript`を使ってブラウザにデータを「キー」と「値」のセットで保存することができる。

  - ローカルストレージ

    | キー | 値 |
    |:---:|:---:|
    | memo | 'さすけちゃん' |

    - `localStorage.setItem('memo', 'さすけちゃん')`: 第１引数に「キー」の名前を、第２引数に「値」を指定する。

    - `localStorage.getItem('memo')`: 引数に「キー」を指定して、ペアになっている「値」を取得する。

    - `localStorage.removeItem('memo')`: 引数に「キー」を指定して、ペアになっている「値」を削除する。

  ```
  # 実装例
    save.addEventListener('click', () => {
      localStorage.setItem('memo', text.value);
    });
  ```
    - 「開発者ツール > アプリケーション > ストレージ > ローカルストレージ > `file://` 」で保存内容を確認できる

- 保存ボタン押下後のメッセージ表示機能の実装例

    ```
    # main.js
      save.addEventListener('click', () => {
        message.classList.add('appear');
        setTimeout(() => {
          message.classList.remove('appear');
        }, 1000);
        localStorage.setItem('memo', text.value);
      });
    ```
    - `setTimeout`で１秒後にメッセージが消える

    ```
    # css
      span {
        opacity: 0;
        transition: opacity 300ms;

        &.appear {
          opacity: 1;
        }
      }
    ```
    -  `transition`は`span`側に記述する。(→ `.appear`が外れるときも`transition`が効く)

  - リロード時に保存データを表示する実装例

    ```
    if (localStorage.getItem('memo') === null) {
      text.value = ``;
    } else {
      text.value = localStorage.getItem('memo');
    }
    ```
    - `getItem()`で取得したデータが`null`だった場合は空文字を表示

- 削除ボタンの実装例
  ```
  # 確認のためのダイアログボックスを表示させる
      clearBtn.addEventListener('click', () => {
        if (confirm('本当に削除しますか？') === true) {
          text.value = '';
          localStorage.removeItem('memo');
        }
      });
  ```

  - 注）削除ボタンの変数名に`delete`は使用できない！

    - JavaScriptのキーワード／予約語は変数名にできない
      (例: `delete`, `class`, `this`などはNG)

    - 英単語1語だけの変数名は予約語と衝突しやすいので要注意
      (`deleteBtn`, `className`, `userThis`などとすればOK)
