- 「カルーセルUI」を試作

  1. 画像を領域内に配置する

    - `overflow: hidden;`: 領域外を非表示にする
    - `object-fit: cover;`: 画像の縦横比を維持する
    - `min-width: 100%;`: 「flex アイテム」を親要素の幅に設定する

  2. ボタンを`container`に対して絶対配置

    - [文字実体参照](https://jmedia.wiki/%25E6%2596%2587%25E5%25AD%2597%25E3%2582%25A8%25E3%2583%25B3%25E3%2583%2586%25E3%2582%25A3%25E3%2583%2586%25E3%2582%25A3/List_of_XML_and_HTML_character_entity_references)
      - `<<`: `&laquo;`と書く
      - `>>`: `&raquo;`と書く

  3. ボタンにクリックイベントを設定

    - `getBoundingClientRect()`: 要素の寸法や相対位置に関する情報を返す
    - 画像の移動は「初期状態からの変移」と捉える
      ```
      # 実装例
        currentIndex++; #「進む」のとき
        currentIndex--; #「戻る」のとき
        const slideWidth = slides[0].getBoundingClientRect().width;
        ul.style.transform = `translateX(${-1 * slideWidth * currentIndex}px)`;
      ```

  4. 画像下のボタンを動的に生成

      ```
      # 実装例
        function setUpDots() {
          for (let i = 0; i < slides.length; i++) {
            const button = document.createElement('button');
            dots.push(button);
            document.querySelector('nav').appendChild(button);
          }

          dots[0].classList.add('current');
        }
      ```

    5. 4のボタンにクリックイベントを設定し、画像内のボタン操作と連動させる

      ```
      # 実装例
        button.addEventListener('click', () => {
          currentIndex = i;
          updateDots();
          hideButtons();
          moveSlides();
        });

      # 連動させたい動作を関数を作って切り出す
        function updateDots() {
          dots.forEach(dot => {
                dot.classList.remove('current');
          });
          dots[currentIndex].classList.add('current');
        }
      ```

    6. 画像幅を広げたときの表示乱れを補正
    
      ```
      # 実装例
        window.addEventListener('resize', () => {
          moveSlides();
        });
      ```
        - ここでは[DOMイベント](https://developer.mozilla.org/ja/docs/Web/API/Document_Object_Model/Events)に`resize`が使われている。
