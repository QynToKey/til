- `Bootstrap`について

  - `container`クラスの[挙動](https://getbootstrap.com/docs/5.3/layout/containers/#how-they-work): 自動でウィンドウ幅に応じてコンテンツを表示する
    - 複数カラムの実装例
    ```
    <div class="container">
      <div class="row">
        <div class="col">
          このカラムを必要数だけ配置する。
        </div>
      </div>
    </div>
    ```
    - グリッドシステム: 1行を12分割して配分を指定する
    - 表示／非表 示を切り替える実装例
    ```
    # デフォルトは非表示／画面幅がlg以上のとき2/12の幅で表示
    <div class="d-none d-lg-block col-lg-2">
    ```

  - 色: docs > Customize > Color > [colors](https://getbootstrap.com/docs/5.3/customize/color/#colors)

  - `margin`や`padding`: Docs > Utilities > [Spacing](https://getbootstrap.com/docs/5.3/utilities/spacing/)
    - `m`:margin, `p`:padding
    - `t':top, `s`:start(=left), `b`:bottom, `e`:end(left)

  - リスト: Docs > Components > List group > [Links and Buttons](https://getbootstrap.com/docs/5.3/components/list-group/#links-and-buttons)

  - テーブル: Docs > Content > [Tables](https://getbootstrap.com/docs/5.3/content/tables/)

  - ボタン: Docs > Components > [Buttons](https://getbootstrap.com/docs/5.3/components/buttons/)

  - フレックスボックス: Docs > Utilities > [Flex](https://getbootstrap.com/docs/5.3/utilities/flex/)

  - アイコン: Icons > [Install](https://icons.getbootstrap.com/#install)

  - ナビゲーション: Docs > Components > [Navbar](https://getbootstrap.com/docs/5.3/components/navbar/)

- `Git` の状況確認
    ```
    git fetch # 情報を取得するだけ(pull は統合まで含む)
    git status
    git log --oneline --graph --decorate --all # どの参照がどのコミットを指しているかを可視化する
    ```
