## RESTful について

- 世の中の「複雑なふるまい」(= **動詞**)を**名詞に変換する**、と考える
  <br>👉 *「ポストする」⇨「ポストを create する」

  | HTTPリクエストメソッド<br>動詞| Path | コントローラー#アクション |
  | --- | --- | --- | --- |
  | GET | /posts | posts#index |
  | GET | /posts/new | posts#new |
  | POST | /posts | posts#create |
  | GET | /posts/:id | posts#show |
  | GET | /posts/:id/edit | posts#edit |
  | PATCH or PUT | /posts/:id | posts#update |
  | DELETE | /posts/:id | posts#destroy |

- **イベントエンティティ**(= 名詞)を見出せば、 `through`で指定して**多対多**の関係をつくれる
  <br>👉 *「コト」を`RESTリソース`とした捉える* ⇨ *`CRUD操作`として表現できる
