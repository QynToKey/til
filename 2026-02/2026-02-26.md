# [卒制](https://github.com/QynToKey/HowLongWillItLast) (day 2)：Rails 初期セットアップ

## gem Sorcery 導入

- `Gemfile`に `gem 'sorcery', '0.16.3'` を記述
- `docker compose exec web bundle install`
- `docker compose exec web rails g sorcery:install`
- `docker compose exec web rails db:migrate`
- `docker compose restart`

### 📝 `run` と `exec`

#### `docker compose run`

- 新しい一時コンテナを作って実行
- 既に動いているコンテナとは別物
- ポートやネットワーク設定が異なることがある
- 状態が分離される可能性あり

#### `docker compose exec`

- 今動いているコンテナの中で実行
- 通常の開発ではこちらが正解
- Railsサーバと同じ環境で実行できる
